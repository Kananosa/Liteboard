# Liteboard
Liteboard is a keyboard with total 40 key,it's super lite and functional (no RGB support)
it absoulely can fit into ur pocket

![3d-main](./picture/3d-main.png "liteboard-3d-picture")

## Features:
- 40 keys 12x4 layout (plank 40%)
- Lite weight and **only 1.3cm thick**
- An volume knob yay
- haptic feedback support by the....wait iPhone haptic engine?
## BOM:
[liteboard-bom.csv]("./bom/liteboard-bom.csv")
MIWU-MINI:an Chinese company shaft switch,i found it on Taobao.
any Low shaft switch i think is okay.

HDR-M_2.54_2x1:a pin header,connecting the haptic engine.
any LRC haptic engine is okay for this keyboard.
